How to Deposit

السلام عليكم, আপনাকে স্বাগতম

  1. প্রথমে আপনি যে মাধ্যমে টাকা পাঠাতে চান (যেমন: নগদ, বিকাশ) সেটি নির্বাচন করুন।
  2. প্রদর্শিত নম্বরে আপনার কাঙ্খিত পরিমাণ টাকা পাঠান।
  3. টাকা পাঠানোর পর, নিচের ফর্মটি পূরণ করুন।
  4. আপনার পাঠানো টাকার মাধ্যম, পরিমাণ, নম্বর এবং Transaction ID (TxID) সঠিকভাবে লিখুন।
  5. সবশেষে "Submit Deposit" বাটনে ক্লিক করুন।
